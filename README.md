# Harrison University Degree Planner
 Java degree planner mobile application for Android.

## Purpose
The primary function of the Degree Planner is to allow students to add, update, and delete terms, courses, course assessments, and course notes in their degree. Details associated with terms, courses, and assessments can be entered manually by the user. The scheduling data is stored in a MySQLite database on the userâ€™s mobile device.
